Instructions

  • This calculator currently does not support imaginary numbers.
  • Use the () convention to input repeating decimals (e.g. 0.6666... = 0.(6)).
  • Use the clear button to clear out all of the cells.
  • Use the copy button to copy the result matrix.
  • Use the paste button to paste the result into the first matrix.
  • Make sure the matrix you intend to paste into the inverse and determinant calculators is a square matrix.
  • Use the O (+/-), R (+/-) or C (+/-) buttons to increase/decrease the order and/or number of rows and columns.
